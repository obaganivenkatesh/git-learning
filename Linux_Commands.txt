            it provides a detailed view of the command which includes Name, Synopsis, Description.
37. cal---> The cal command displays a calendar of the specified year or month.
38. echo--->echo command in linux is used to display line of text/string that are passed as an argument . This is a built in command that is mostly used in shell scripts and batch files to output status text to the screen or a file.
39. userdel--->userdel command in Linux system is used to delete a user account and related files. This command basically modifies the system account files, deleting all the entries which refer to the username LOGIN.
